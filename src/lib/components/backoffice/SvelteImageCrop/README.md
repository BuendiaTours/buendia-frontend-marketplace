# SvelteImageCrop

Image crop editor component for SvelteKit + Svelte 5.

All available props, visibility options, output config, events, and public methods are documented in:

**[`types/imageEditorTypes.ts`](./types/imageEditorTypes.ts)**

Key types:

- `SicImageEditorProps` — component props
- `UIVisibility` — controls which UI elements are shown
- `OutputConfig` — generated crop format/quality settings
- `SicImageEditorEvents` — events emitted by the editor
- `SicImageEditorInstance` — public methods (via `bind:this`)
